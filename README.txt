== Description ==
Not everyone wants to see 'Howdy, <username>' on their WordPress admin screen. This allows you to customize it to whatever you choose.


Requires PHP 5

== Installation ==

Just click Install in the WordPress plugins directory.
Activate the plugin in the plugins menu.
Navigate to Settings --> Custom Howdy, enter your text, and watch the magic!
